# 📚 Projeto 3: Biblioteca Triple Peaks

A página da **Biblioteca Triple Peaks** é o terceiro projeto do programa de Desenvolvimento Web da TripleTen.

O projeto foi desenvolvido com **HTML e CSS**, seguindo um roteiro mais guiado, com foco na estruturação semântica do HTML e na criação de layouts utilizando Flexbox.

---

## 🎯 Objetivo do projeto

- Praticar o uso de **HTML5 semântico**
- Criar layouts utilizando **Flexbox**
- Trabalhar **posicionamento de elementos**
- Consolidar a estrutura básica de uma página web estática

---

## 🛠️ Recursos e técnicas aplicadas

- HTML5 semântico
- Flexbox
- Posicionamento de elementos

---

## 📦 Como rodar o projeto localmente

- Clone o repositório:
```bash
git clone git@github.com:VanessaYuriAB/web_project_library_pt.git
cd web_project_library_pt
```

- Este é um projeto estático (_HTML e CSS_). Para visualizar:
  - Abra o arquivo `index.html` no navegador  
  **ou**
  - Use uma extensão como `Live Server` no `VS Code`
